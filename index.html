📌 طريقة الاستخدام

  1. ألصق المحتوى وحدد عدد كل نوع من الأسئلة (يمكن وضع 0 لتجاهل النوع)
  2. فعّل خيارات الترتيب العشوائي والصعوبة حسب رغبتك
  3. اضغط "أنشئ البرومبت" ثم "نسخ البرومبت"
  4. افتح chatgpt.com أو gemini.google.com وألصق البرومبت
  5. انسخ الرد كاملاً وألصقه في الخانة الثانية
  6. اختر صيغة الإخراج (QTI موصى به لـ Ultra)
  7. اضغط "حوّل وأنزل الملف"

📚 رفع QTI 2.1 على بلاك بورد Ultra

  1. اذهب إلى محتوى المقرر
  2. في اللوحة الجانبية → Manage Banks أو إدارة بنوك الأسئلة
  3. اضغط + ← Import from File
  4. اختر ملف blackboard_QTI_*.zip ✅

📄 رفع TXT داخل اختبار

  1. اذهب إلى محتوى المقرر ← + ← Test
  2. داخل الاختبار، اضغط + ← رفع أسئلة من ملف
  3. اختر ملف blackboard_questions_*.txt ✅
